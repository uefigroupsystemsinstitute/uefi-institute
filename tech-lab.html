# UEFI Module: AI Prompt Formatting Pipeline

def format_ai_prompt(user_input: str) -> str:
    # 1. Sanitize text (remove trailing spaces)
    clean_text = user_input.strip().lower()
    
    # 2. System Guardrail Enforcement
    if not clean_text.isalnum():
        clean_text = clean_text.replace("!", "")
        
    # 3. f-string injection for LLM context
    system_role = "You are an Azure Cloud Architect."
    prompt = f"[SYSTEM]: {system_role}\n[USER]: {clean_text}"
    
    return prompt

print(format_ai_prompt(" Deploy a scalable DB! "))
# Output: [SYSTEM]: You are an Azure Cloud Architect.
# Output: [USER]: deploy a scalable db

The 4-Shot AI Prompting Architecture

1

System Contextualization

Bind the LLM to a strict persona (e.g., "Act as a Senior Database Engineer") to prevent hallucination.

2

Few-Shot In-Context Learning

Provide JSON or Python code examples *inside* the prompt so the AI learns the required output format instantly.

3

Chain-of-Thought (CoT) Strategy

Force the AI to output <thought_process> before the final answer to guarantee logical math/code generation.

4

Schema Guardrails

Explicitly deny markdown formatting if building an API ("Return ONLY raw JSON, do not use ```json wrappers").

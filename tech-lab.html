raw_prompt = "   analyze this dataset for anomalies in Nairobi market prices  "

# 1. Clean whitespace
clean_prompt = raw_prompt.strip()

# 2. Modern f-string formatting
formatted_input = f"System Prompt: [ROLE: AI Analyst] | User: {clean_prompt.upper()}"

# 3. String slicing & checks
is_valid = formatted_input.startswith("System Prompt")

print(f"Cleaned: {clean_prompt}")

print(f"Valid: {is_valid}")

4 Pillars of AI Prompt Engineering

1. Role & Context Framing

Assign explicit personas (e.g., "Act as a Senior Microsoft Azure Architect") to anchor LLM responses in specialized domain knowledge.

2. Few-Shot In-Context Learning

Provide 2-3 structured input-output exemplars inside the prompt to enforce precision formatting (JSON, SQL schemas, Excel formulas).

3. Chain-of-Thought (CoT) Reasoning

Instruct the model to "Think step-by-step" before producing final code to minimize hallucinations in complex math and logic.

4. Output Constraints & Schema Guardrails

Specify explicit rules (e.g., "Respond ONLY in raw JSON format without markdown blocks") for reliable API integration.
